# EvoAML: 演化反洗钱监管科技框架

[English](README.md) | [中文](README_zh.md)

## 🎯 Proposed Endeavor (核心主张)
开发并推进一个融合图网络跨行业监控与时序行为演化分析的智能AML检测框架（EvoAML），旨在弥合当前反洗钱基础设施在跨行业资金流追踪和动态洗钱模式识别上的系统性技术缺口，并将该方法论转化为符合美国监管框架（BSA/AMLA 2020）的可部署解决方案。

## 💡 核心架构
EvoAML 致力于填补最前沿的AI学术研究与实际合规监管科技落地之间的鸿沟。
1. **图驱动跨行业追踪**: 采用图神经网络 (GNN) 在复杂的供应链和新能源移动领域追踪隐蔽的资金洗钱路径。
2. **时序演化分析**: 利用动态自注意力网络预测和识别随时间突变的洗钱行为模式。
3. **AMLA 2020 合规引擎**: 将 AI 检测到的异常自动转化为符合《银行保密法》(BSA) 标准的可疑活动报告 (SAR) 文本格式。

## 📅 项目路线图 (六阶段计划)
详细的技术拆解请参考 [development_phases.md](docs/planning/development_phases.md)。

- **第一阶段:** 基础设施与合规架构设计 ✅ (当前阶段完成)
- **第二阶段:** 异构数据摄取与预处理流水线
- **第三阶段:** 图驱动追踪模块集成 (GNN 模型)
- **第四阶段:** 时序行为演化分析模块集成
- **第五阶段:** BSA/AMLA 2020 合规引擎与 SAR 报告自动生成
- **第六阶段:** 系统级业务模拟、可视化看板与 v1.0 发布
