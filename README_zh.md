# EvoAML: 演化反洗钱监管科技框架

[English](README.md) | [中文](README_zh.md)

## 概览
EvoAML 是一个开源的监管科技框架，旨在弥合前沿人工智能研究与实际反洗钱合规应用之间的差距。它提供了一个模块化的管道，用于检测跨行业金融网络中的复杂洗钱模式。

## 核心架构
EvoAML 集成多种检测方法以提供全面覆盖：

1. **图驱动跨行业追踪** [开发中]
   *使用基于图的分析来追踪供应链和多部门网络中的资金流。*

2. **时序演化分析** [开发中]
   *应用时间序列分析来识别表明洗钱的演变行为模式。*

3. **AMLA 2020 合规引擎** [已实现]
   *将检测结果转化为符合 BSA 标准的 SAR（可疑活动报告）叙述文本。*

## 快速开始
1. 安装依赖: `pip install -r requirements.txt`
2. 启动看板: `streamlit run app.py`
3. 参考 `examples/` 目录下的算法工作流。
